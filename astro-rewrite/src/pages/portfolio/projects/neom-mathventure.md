Neom Mathventure is an interactive educational game that helps children learn math concepts through adventure.

## Key Features
- **Gamified Learning**: Math problems integrated into gameplay.
- **Adaptive Difficulty**: Adjusts to the player's skill level.
- **Mascots**: Featuring Thangamma the Tapir and Kannappan the Capybara.
- **Voice Interaction**: Uses AI to recognize speech for answering questions.
- **Performance**: Built with Rust and WebAssembly for high performance.

Check it out on [GitHub](https://github.com/nrynss/neom-mathventure).
